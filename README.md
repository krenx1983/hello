# 即时通讯客户端

十年前写的一个即时通讯系统，写了玩的，客户端准备开源出来，有兴趣的可以关注产品的进展。

绿色免安装，只有一个exe文件。

无需注册即可登录，用户名可以随便输入，密码可以没有。

如果需要保护用户名，可以注册。

支持好友聊天、群聊、文件/文件夹传输、微博、论坛、便签、模拟期货、小会计等等功能。

也支持消息漫游访问（换一台电脑，也可以收取之前所有的聊天内容。）。

支持一个用户多客户端同时在线，可同时收到同样的聊天信息。
